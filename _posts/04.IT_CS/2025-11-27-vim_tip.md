## 마킹하기

```nvim
ma # 돌아올 곳에 마킹하기 (a말고 다른 레지스터에 저장해도 됨)

'a # 돌아가기
```


## 매크로

```nvim 
qa # 매크로 녹화
@a # 매크로 실행


@@ # 이전 매크로 실행
```



## 여러 숫자 한번에 바꾸기

```nvim
1. abc
1. def
1. ghi
1. jkl
```

위의 앞에 번호의 순서를 '1. 2. 3. 4.'로 바꿀 것이다.

```nvim
1. 각 번호를 ctrl + v로 드래그한다.
2. g
3. ctrl + a 

```
이렇게 하면 번호가 순서대로 증가하게 된다.




## 파일전체에서 텍스트 한번에 바꾸기

```nvim
:%s/변경될 텍스트/변경할 텍스트/gc # :3s/~ 이렇게 하면 3행에서 대체
```



## 특정 레지스터에 동작 저장 및 실행
```nvim
"a #a 레지스터에 동작 저장
"+ # 이건 시스템 클립보드에 저장하는 특수 레지스터다
```
