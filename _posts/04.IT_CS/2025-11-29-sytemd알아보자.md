## systemctl 명령어#

```bash
systemctl [옵션] [명령] [유닛]
```

### 옵션
-a : 유닛 전체 출력
-t 유닛: 지정한 유닛만 출력


### 명령
1. status
2. start
3. stop 
4. restart

1. enable
2. disable

1. reload

> status , start, enable

> User, ExecStart, Environment
- Type = 서비스 타입(주로 simple 사용)
- User = 어떤 유저 권한을 실행할지
- WorkingDirectory = 명령어를 실행할 기본 디렉터리
- ExecStart = 실행 명령어
- Environment
- Restart

#### Install 섹션
