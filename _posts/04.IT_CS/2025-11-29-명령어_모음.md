## awk 명령어#

```bash 
$ awk 'BEGIN { print "~~"} {print $1, $2} END {print "~"}' file.txt
```


## tmux 명령어#

```bash 
tmux ls : 현재 세션 목록 나열
tmux 실행 중 ctrl+b -> d 를 누르고 현재 세션에서 빠져나오기
tmux a -t 세션 이름 : 특정 세션으로 들어가기


```
